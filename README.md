# Cafe Management GUI Application

## Overview

A desktop-based cafe management application developed using Java Swing. The application provides graphical interfaces for user authentication, menu browsing, order management, and reservation handling.

## Features

- User registration and login
- Food menu viewing
- Beverage menu viewing
- Additional menu viewing
- Order creation and management
- Pending order tracking
- Order history tracking
- Reservation management
- Reservation information display
- Multi-window graphical user interface

## Technologies Used

- Java
- Java Swing
- NetBeans GUI Builder
- Object-Oriented Programming (OOP)

## Main Modules

| Module | Description |
|----------|----------|
| Sign In | User authentication |
| Sign Up | User registration |
| Home | Main dashboard |
| Input Order | Create and manage orders |
| Order History | View previous orders |
| Order Pending | View pending orders |
| Reserve Order | Create reservations |
| Reserve Info | View reservation details |
| Menu Pages | Display food, beverage, and additional menus |

## Requirements

- Java Development Kit (JDK 8 or later)
- NetBeans IDE (recommended)

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/Cafe-Management-GUI.git
```

2. Open the project in NetBeans IDE.

3. Build the project.

4. Run:

```text
CafeGUI.java
```
